## 9.1 视频标签（video）

video标签用于在页面中嵌入视频，是HTML5新增的双标签。

### 9.1.1 基本属性

```html
<video src="./movie.mp4" controls width="800" poster="./cover.jpg">
    您的浏览器不支持video标签
</video>
```

各属性说明：

**src** — 视频文件的URL地址

**width/height** — 设置视频播放器的宽高（像素值）。只设置其中一个时，另一个按比例计算。

**controls** — 布尔属性，显示浏览器内置的播放控件（播放/暂停、进度条、音量、全屏等）。不设置则只显示第一帧画面。

**autoplay** — 布尔属性，页面加载后自动播放。现代浏览器通常要求同时设置muted才允许自动播放（用户未主动交互前不允许有声自动播放，这是浏览器的自动播放策略）。

**muted** — 布尔属性，视频静音。

**loop** — 布尔属性，循环播放。

**poster** — 视频封面图片URL，视频加载前或未播放时显示。

**preload** — 视频预加载策略：
- none — 不预加载，用户点击播放时才开始加载，最节省带宽
- metadata — 仅预加载元数据（时长、尺寸等），不下载视频内容（默认值）
- auto — 浏览器可以自行决定是否预加载整个视频（使用autoplay时，此属性被忽略）

标签之间的内容是降级文本，当浏览器不支持video标签时会显示。

### 9.1.2 多格式兼容

不同浏览器支持的视频格式不同，可以通过source标签提供多种格式作为备选：

```html
<video controls width="800">
    <source src="./movie.mp4" type="video/mp4">
    <source src="./movie.webm" type="video/webm">
    <source src="./movie.ogg" type="video/ogg">
    您的浏览器不支持video标签
</video>
```

浏览器会按顺序尝试加载，使用第一个能支持的格式。

常用视频格式：
- MP4（H.264）— 兼容性最好，推荐优先使用
- WebM（VP8/VP9）— 开源格式，Chrome/Firefox支持良好
- Ogg（Theora）— 开源格式，Safari不支持

## 9.2 音频标签（audio）

audio标签用于在页面中嵌入音频，是HTML5新增的双标签。用法与video类似，但没有width、height、poster属性（音频没有画面）。

```html
<audio src="./music.mp3" controls loop>
    您的浏览器不支持audio标签
</audio>
```

各属性说明：

**src** — 音频文件URL

**controls** — 显示播放控件（播放/暂停、进度条、音量）

**autoplay** — 自动播放（同样需要配合muted）

**muted** — 静音

**loop** — 循环播放

**preload** — 预加载策略（同video：none/metadata/auto）

多格式兼容写法：

```html
<audio controls>
    <source src="./music.mp3" type="audio/mpeg">
    <source src="./music.ogg" type="audio/ogg">
    <source src="./music.wav" type="audio/wav">
    您的浏览器不支持audio标签
</audio>
```

常用音频格式：
- MP3 — 兼容性最好
- Ogg — 开源格式
- WAV — 无损格式，文件体积大
