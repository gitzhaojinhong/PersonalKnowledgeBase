# 十六、非兼容性改变

Vue 3 相对于 Vue 2 的一些不兼容变更，升级时需要注意：

1. **过渡类名变化**
   - `v-enter` → `v-enter-from`
   - `v-leave` → `v-leave-from`

2. **v-on 修饰符变化**
   - 移除了 keyCode 作为 v-on 修饰符的支持

3. **v-model 变化**
   - v-model 指令在组件上的使用已被重新设计
   - 替换掉了 Vue 2 的 v-bind.sync

4. **v-if 和 v-for 优先级**
   - v-if 和 v-for 在同一个元素上使用时，优先级发生了变化

5. **移除的实例方法**
   - 移除了 `$on`、`$off`、`$once` 实例方法

6. **移除过滤器**
   - 移除了过滤器（filter）

7. **移除 $children**
   - 移除了 `$children` 实例属性
