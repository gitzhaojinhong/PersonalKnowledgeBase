## 5.1 v-bind 动态属性绑定

`v-bind` 用于**动态绑定 HTML 属性到 Vue 实例的数据上**，**值是 JavaScript 表达式**。

### 5.1.1 基本用法

```vue
<!-- 完整语法 -->
<img v-bind:src="imageSrc">

<!-- 简写语法（推荐） -->
<img :src="imageSrc">
```

### 5.1.2 绑定字符串

```vue
<template>
  <a :href="url">点击这里</a>
</template>
<script lang="ts" setup>
  const url = "https://vuejs.org";
</script>
```

### 5.1.3 绑定布尔值

```vue
<template>
  <button :disabled="isButtonDisabled">提交</button>
</template>
<script lang="ts" setup>
  const isButtonDisabled = true;
</script>
```

> **注意：** 如果变量值是 `null` 或 `undefined`，该属性不会渲染（属性不存在）。

### 5.1.4 绑定数组（动态 class）

**知识点：**

1. `:class="[activeClass, errorClass]"` 是 Vue 对 `class` 属性的特殊处理语法。
2. 如果 `activeClass` 和 `errorClass` 两个变量都有值，渲染结果为：`class="active text-danger"`

```vue
<template>
    <div :class="[activeClass, errorClass]">
        这是一个带有动态类名的 div 元素
    </div>
    <!-- activeClass='active', errorClass='text-danger' 时，渲染为： -->
    <!-- <div class="active text-danger"> -->
    <!-- activeClass='active', errorClass='' 时，渲染为： -->
    <!-- <div class="active"> -->
</template>
<script lang="ts" setup>
    const activeClass = 'active';
    const errorClass = 'text-danger';
</script>
<style scoped>
    .active {
        background-color: #42b983;
        border: 1px solid #3aa876;
    }
    .text-danger {
        color: #f56c6c;
        font-weight: bold;
    }
</style>
```

### 5.1.5 绑定对象（动态 class）

**知识点：**

1. `:class="{ 'active': isActive, 'text-danger': hasError }"` 是 Vue 的特殊语法。
2. 如果 `isActive` 和 `hasError` 都是 `true`，渲染结果：`class="active text-danger"`

```vue
<template>
    <div :class="{ 'active': isActive, 'text-danger': hasError }">
        这是一个带有动态类名的 div 元素
    </div>
</template>
<script lang="ts" setup>
    const isActive = true;
    const hasError = true;
</script>
<style scoped>
    .active {
        background-color: #42b983;
        color: white;
        border: 1px solid #3aa876;
    }
    .text-danger {
        color: #f56c6c;
        font-weight: bold;
    }
</style>
```

### 5.1.6 动态属性名

```vue
<template>
    <div :[attributeName]="value"></div>
    <!-- 渲染结果：<div id="main-content"></div> -->
</template>
<script lang="ts" setup>
    const attributeName = 'id';
    const value = 'main-content';
</script>
```

### 5.1.7 一次性绑定多个属性

```vue
<template>
    <div v-bind="objectOfAttrs"></div>
    <!-- 简写 -->
    <div :="objectOfAttrs"></div>
    <!-- 渲染结果：<div id="container" class="wrapper" data-attribute="example"></div> -->
</template>
<script lang="ts" setup>
    const objectOfAttrs = {
        id: 'container',
        class: 'wrapper',
        'data-attribute': 'example'
    };
</script>
```

### 5.1.8 综合应用示例

```vue
<template>
    <div>
        <img :src="imagePath" :alt="imageDescription">
        <div :style="{ color: textColor, fontSize: fontSize + 'px' }">动态样式文本</div>
        <input type="text" :value="inputValue">
        <button :class="{ 'btn-primary': isPrimary, 'btn-disabled': isDisabled }" :disabled="isDisabled">
            点击我
        </button>
    </div>
</template>
<script lang="ts" setup>
    import { ref } from 'vue';

    const imagePath = '/vite.svg';
    const imageDescription = '示例图片';
    // ref() 函数不是必须的，需要响应式就加，不需要响应式可以不加
    const textColor = ref('#42b983');
    const fontSize = ref(16);
    const inputValue = ref('jackson');
    const isPrimary = ref(true);
    const isDisabled = ref(false);
</script>
<style scoped>
    .btn-primary { background-color: #42b983; color: white; }
    .btn-primary:hover { background-color: #3aa876; transform: translateY(-2px); }
</style>
```

## 5.2 v-model 双向数据绑定

`v-model` 指令可以完成**双向数据绑定**效果。

**什么是双向数据绑定？**
1. **数据修改时视图变化。**
2. **视图修改时数据也变化。**

### 5.2.1 手动实现双向绑定（原理说明）

```vue
<template>
    <div>
        <!-- @input：发生 input 事件时执行后面的表达式 -->
        <!-- $event：当前发生的事件 -->
        <!-- $event.target：当前发生事件的目标元素 -->
        <!-- as HTMLInputElement：TypeScript 断言，解决编译报错 -->
        <input type="text" :value="inputValue"
               @input="inputValue = ($event.target as HTMLInputElement).value">
        <div>您输入的用户名为：{{ inputValue }}</div>
    </div>
</template>
<script lang="ts" setup>
    import { ref } from 'vue';
    const inputValue = ref('jackson');
</script>
```

### 5.2.2 使用 v-model（推荐）

```vue
<template>
    <div>
        <!-- v-model 自动完成双向数据绑定 -->
        <!-- 注意：不是 v-model:value，而是 v-model -->
        <input type="text" v-model="inputValue">
        <div>您输入的用户名为：{{ inputValue }}</div>
    </div>
</template>
<script lang="ts" setup>
    import { ref } from 'vue';
    const inputValue = ref('jackson');
</script>
```

## 5.3 v-for 列表渲染

### 5.3.1 基本语法

```vue
<template>
    <ul>
        <li v-for="person in personList" :key="person.id">
            {{ person.name }}, {{ person.age }}
        </li>
    </ul>
</template>
<script lang="ts" setup>
    let personList = [
        { id: '101', name: 'jack', age: 20 },
        { id: '102', name: 'lucy', age: 30 },
        { id: '103', name: 'tom', age: 25 }
    ];
</script>
```

`v-for` 的基本语法：`item in items`，其中 `items` 是源数据数组，`item` 是数组元素别名。

**重要：** 始终提供唯一的 `:key` 属性，帮助 Vue 识别节点，提高性能，避免潜在渲染问题。通常使用数据中的唯一 ID。

### 5.3.2 获取索引

```vue
<template>
    <ul>
        <li v-for="(person, index) in personList" :key="person.id">
            {{ index }}, {{ person.name }}, {{ person.age }}
        </li>
    </ul>
</template>
<script lang="ts" setup>
    let personList = [
        { id: '101', name: 'jack', age: 20 },
        { id: '102', name: 'lucy', age: 30 },
    ];
</script>
```

### 5.3.3 遍历对象

```vue
<template>
    <ul>
        <!-- (值, 键名, 索引) -->
        <li v-for="(propertyValue, propertyName, index) in person" :key="propertyName">
            {{ index }}, {{ propertyName }}: {{ propertyValue }}
        </li>
    </ul>
</template>
<script lang="ts" setup>
    let person = { name: 'jackson', age: 20, tel: '13000232124' };
</script>
```

### 5.3.4 使用范围值

```vue
<template>
    <ul>
        <!-- 渲染 1 到 10 -->
        <li v-for="n in 10" :key="n">{{ n }}</li>
    </ul>
</template>
```

## 5.4 条件渲染指令

### 5.4.1 v-else

```vue
<template>
  <div v-if="isLoggedIn">欢迎回来！</div>
  <div v-else>请先登录</div>
</template>
<script setup>
    import { ref } from 'vue';
    const isLoggedIn = ref(false);
</script>
```

### 5.4.2 v-else-if

```vue
<template>
  <div v-if="score >= 90">优秀</div>
  <div v-else-if="score >= 60">及格</div>
  <div v-else>不及格</div>
</template>
<script setup>
    import { ref } from 'vue';
    const score = ref(85);
</script>
```

### 5.4.3 v-if 与 v-show 的区别

这两个指令都可以控制组件的显示和隐藏，`true` 显示，`false` 隐藏：

1. `v-if` 隐藏时会**彻底删除**组件（DOM 中不存在）。
2. `v-show` 隐藏时只是把 `display` 样式修改为 `none`，**不会删除组件**（DOM 中仍存在）。

```vue
<template>
    <h3 v-if="flag1">v-if 指令（隐藏时删除 DOM）</h3>
    <h3 v-show="flag2">v-show 指令（隐藏时仅 display:none）</h3>
</template>
<script lang="ts" setup>
    import { ref } from 'vue';
    let flag1 = ref(true);
    let flag2 = ref(true);
</script>
```

## 5.5 其他实用指令

### 5.5.1 v-text

设置元素的 `textContent`，相当于 `{{ }}`，但会覆盖整个内容。

```vue
<template>
  <div v-text="message"></div>
  <!-- 等同于 <div>{{ message }}</div> -->
</template>
<script setup>
    import { ref } from 'vue';
    const message = ref("Hello Vue 3!");
</script>
```

### 5.5.2 v-html

渲染原始 HTML，**注意：动态 HTML 可能有 XSS 风险，仅用于可信内容。**

```vue
<template>
  <div v-html="rawHtml"></div>
</template>
<script setup>
    import { ref } from 'vue';
    const rawHtml = ref('<span style="color: red">红色文字</span>');
</script>
```

### 5.5.3 v-pre

跳过编译，直接输出原始内容（适用于展示 Vue 模板代码）。

```vue
<template>
  <div v-pre>{{ 这个内容不会被编译 }}</div>
  <!-- 渲染结果：{{ 这个内容不会被编译 }} -->
</template>
```

### 5.5.4 v-cloak

防止 Vue 未编译完成时显示 `{{ }}` 模板（需配合 CSS 使用）。

```vue
<template>
  <div v-cloak>{{ message }}</div>
</template>
<script lang="ts" setup>
    import { ref } from 'vue';
    let message = ref("hello message!");
</script>
<style>
    [v-cloak] {
        display: none;
    }
</style>
```

### 5.5.5 v-once

仅渲染一次，后续数据变化不会更新（适用于静态内容优化）。

```vue
<template>
  <div v-once>{{ staticMessage }}</div>
  <!-- 即使 staticMessage 变化，这里也不会更新 -->
</template>
<script setup>
    import { ref } from 'vue';
    const staticMessage = ref("这段内容只渲染一次");
</script>
```

### 5.5.6 v-memo（Vue 3 新增）

> **原理：** 组件内任何在模板中使用的响应式数据变化 → 触发组件重新渲染 → 重新执行 render 函数 → 所有模板中的函数都会重新调用。

**作用：** 缓存模板片段，仅在依赖项变化时重新渲染（优化性能）。

```vue
<template>
  <button @click="data++">其他组件更新 {{ data }}</button>
  
  <!-- 有 v-memo：点击按钮不会重新计算（因为 dependency1/dependency2 没变） -->
  <div v-memo="[dependency1, dependency2]">
    有缓存：{{ expensiveCalculation() }}
  </div>
</template>
<script setup>
  import { ref } from 'vue';
  const dependency1 = ref(1);
  const dependency2 = ref(2);
  const data = ref(0);
  const expensiveCalculation = () => {
    console.log("计算中...");
    return dependency1.value + dependency2.value;
  };
</script>
```

### 5.5.7 指令总结

| 指令 | 作用 | 适用场景 |
| --- | --- | --- |
| `v-else` | `v-if` 的"否则"情况 | 条件渲染 |
| `v-else-if` | 多条件分支 | 复杂条件判断 |
| `v-text` | 设置 `textContent` | 替代 `{{ }}` |
| `v-html` | 渲染原始 HTML | 动态 HTML（需防 XSS） |
| `v-pre` | 跳过编译 | 展示 Vue 模板代码 |
| `v-cloak` | 防止未编译时显示 `{{ }}` | 配合 CSS 使用 |
| `v-once` | 仅渲染一次 | 静态内容优化 |
| `v-memo` | 缓存模板片段 | 性能优化 |
