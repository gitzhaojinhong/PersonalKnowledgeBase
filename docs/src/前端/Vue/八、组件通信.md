
# 八、组件通信

一个项目中，组件会有很多，组件通信指的是组件之间数据的传递。

## 8.1 props（父子通信）

`props` 是组件通信中最常用的一种方式，适合**父子之间互相传递数据**。不太适合爷孙或更多层级。

- **父传子**：属性值是非函数。
- **子传父**：属性值是函数（子调用父传过来的函数，将数据作为参数传入）。

**父组件：**

```vue
<template>
    <div class="parent">
        <h1>父组件</h1>
        <h2 v-if="fromSon">收到了子的：{<!-- -->{ fromSon }}</h2>
        <!-- :life 是父传子（数据），:acceptLove 是子传父（函数） -->
        <Son :life="life" :acceptLove="acceptLove"/>
    </div>
</template>
<script lang="ts" setup>
    import { ref } from 'vue';
    import Son from './Son.vue';

    let life = ref('生命');
    let fromSon = ref('');

    function acceptLove(value: string) {
        fromSon.value = value;  // 子组件通过调用此函数，将数据传递给父组件
    }
</script>
```

**子组件：**

```vue
<template>
    <div class="son">
        <h2>子组件</h2>
        <h3>父给了我：{<!-- -->{ life }}</h3>
        <button @click="acceptLove('关爱')">我应该关爱我的父</button>
    </div>
</template>
<script lang="ts" setup>
    // defineProps 接收父组件传递过来的数据
    // 注意：defineProps() 的参数必须是一个数组，即使只有一个参数也需要写成数组形式
    // 接收后在模板中直接可用
    defineProps(['life', 'acceptLove']);
</script>
```

### 8.1.1 传递静态数据

```vue
<!-- 父组件 -->
<Person name='jack' age="18"/>

<!-- 子组件 -->
<script lang="ts" setup>
    // defineProps 返回值是一个对象，包含所有接收到的数据
    const fromParentDataObj = defineProps(['name', 'age']);
    console.log(fromParentDataObj.name, fromParentDataObj.age);
</script>
```

### 8.1.2 传递动态数据

```vue
<!-- 父组件：使用 v-bind 传递动态数据 -->
<Person v-bind:name='nameData' :age="ageData"/>

<script lang="ts" setup>
    import Person from './components/Person.vue';
    let nameData = '杰克';
    let ageData = 19;
</script>
```

### 8.1.3 传递数组

```vue
<!-- 父组件 -->
<template>
    <Person :list="personList"/>
</template>
<script lang="ts" setup>
    import Person from './components/Person.vue';
    import { reactive } from 'vue';
    let personList = reactive([
        { id: '101', name: 'jack', age: 18 },
        { id: '102', name: 'lucy', age: 19 },
        { id: '103', name: 'tom', age: 20 }
    ]);
</script>

<!-- 子组件 -->
<template>
    <ul>
        <li v-for="p in list" :key="p.id">
            {<!-- -->{ p.id }}, {<!-- -->{ p.name }}, {<!-- -->{ p.age }}
        </li>
    </ul>
</template>
<script lang="ts" setup>
    defineProps(['list']);
</script>
```

### 8.1.4 props 类型限定

```vue
<!-- 子组件：使用 TypeScript 泛型限定类型 -->
<script lang="ts" setup>
    interface Person {
        id: number | string,
        name: string,
        age: number
    }

    // 泛型约束：父组件必须传递 Person[] 数组
    defineProps<{ list: Person[] }>();

    // 可选 props（可传可不传）
    // defineProps<{ list?: Person[] }>();
</script>
```

### 8.1.5 指定默认值（withDefaults）

```vue
<script lang="ts" setup>
    interface Person {
        id: number | string,
        name: string,
        age: number
    }

    // 如果父组件没有传 list，则使用默认值
    withDefaults(defineProps<{ list?: Person[] }>(), {
        list: () => [
            { id: '111', name: '张三', age: 20 },
            { id: '222', name: '李四', age: 21 },
            { id: '333', name: '王五', age: 22 }
        ]
    });
</script>
```

## 8.2 自定义事件（子向父）

自定义事件只适合**子向父传递数据**。

原理：在**父组件**中给**子组件标签**绑定一个**自定义事件**，然后在**子组件**中调用 `emit` 方法来触发这个事件，通过 `emit` 方法来传递数据。

> **Vue 开发规范：** 自定义事件的名字要遵循 **kebab-case** 风格（烤肉串式命名）。

**父组件：**

```vue
<template>
    <div class="parent">
        <h1>父组件</h1>
        <h3 v-if="fromSon">收到了子的：{<!-- -->{ fromSon }}</h3>
        <!-- 绑定自定义事件 get-love -->
        <Son @get-love="acceptLove"/>
    </div>
</template>
<script lang="ts" setup>
    import { ref } from 'vue';
    import Son from './Son.vue';

    let fromSon = ref('');

    // 触发 get-love 事件时，传递过来的数据会自动注入到 acceptLove 函数中
    function acceptLove(loveFromSon: string, event: Event) {
        fromSon.value = loveFromSon;
        console.log('发生事件的目标元素是：', event.target);
    }
</script>
```

**子组件：**

```vue
<template>
    <div class="son">
        <h2>子组件</h2>
        <!-- $event 是固定写法，表示当前事件对象 -->
        <button @click="emit('get-love', '关爱', $event)">我应该关爱我的父</button>
    </div>
</template>
<script lang="ts" setup>
    // defineEmits() 用来声明/定义事件
    const emit = defineEmits(['get-love']);
</script>
```

## 8.3 mitt（任意组件通信）

### 8.3.1 mitt 概述

1. `mitt` 可以完成**任意组件之间**的通信，推荐仅在需要**全局事件**时使用。
2. `mitt` 是微型事件总线（仅 200 bytes），超轻量级，无依赖，支持 TypeScript，是 Vue 3 中替代 Event Bus 的首选方案。
3. **实现原理**：用一个 `Map` 存储事件名和对应的回调函数列表，通过 `.on()` 订阅时存入回调，`.emit()` 发布时遍历执行对应回调，`.off()` 移除指定回调。本质是**发布-订阅模式的极简实现**。

### 8.3.2 安装 mitt

```bash
npm i mitt
```

### 8.3.3 暴露 emitter

创建 `utils/emitter.ts`：

```typescript
import mitt from "mitt";

// 调用 mitt() 方法返回 emitter，emitter 可以：绑定事件、触发事件
const emitter = mitt();

/*
// 绑定事件
emitter.on('event1', () => { console.log('event1 事件触发了'); });
emitter.on('event2', () => { console.log('event2 事件触发了'); });

// 触发事件
setInterval(() => {
    emitter.emit('event1');
    emitter.emit('event2');
}, 1000);

// 解绑事件
setTimeout(() => {
    emitter.off('event1');   // 解绑 event1
    emitter.all.clear();     // 解绑所有事件
}, 3000)
*/

export default emitter;
```

### 8.3.4 实现兄弟组件间的数据传递

**父组件：**

```vue
<template>
    <div class="parent">
        <h1>父组件</h1>
        <Son1></Son1>
        <Son2></Son2>
    </div>
</template>
<script lang="ts" setup>
    import Son1 from './Son1.vue';
    import Son2 from './Son2.vue';
</script>
```

**子组件 1（发送方 — 触发事件）：**

```vue
<template>
    <div class="son">
        <h2>子组件1</h2>
        <button @click="giveMoney">给弟弟一万元</button>
    </div>
</template>
<script lang="ts" setup>
    import emitter from '../utils/emitter';

    function giveMoney() {
        emitter.emit('send-money', 10000);  // 触发事件，传递数据
    }
</script>
```

**子组件 2（接收方 — 绑定事件）：**

```vue
<template>
    <div class="son">
        <h2>子组件2</h2>
        <h3 v-if="money">哥哥给了我 {<!-- -->{ money }} 元</h3>
    </div>
</template>
<script lang="ts" setup>
    import { ref, onUnmounted } from 'vue';
    import emitter from '../utils/emitter';

    let money = ref();

    // 绑定事件，接收数据
    emitter.on('send-money', (value) => {
        money.value = value;
    });

    // 组件卸载时，建议解除事件绑定（避免内存泄漏）
    onUnmounted(() => {
        emitter.off('send-money');
    });
</script>
```

## 8.4 ref 与 defineExpose（子向父）

### 8.4.1 普通标签上使用 ref

现代开发方式都是 SPA 单网页应用，多个 Vue 组件最终渲染到同一个 HTML 文件中，因此**不要使用 `id` 属性**标记元素（不同组件中可能出现相同的 `id`，会产生覆盖和干扰）。使用 `ref` 属性代替。

**注意：** 只有在**页面渲染之后**，脚本中的 `myDiv` 才会有值。渲染前是 `undefined`。

```vue
<!-- 子组件 Person.vue -->
<template>
    <div ref='myDiv'>这是 Person 组件中的一个 div 元素</div>
    <button @click="showDiv">输出 div 元素</button>
</template>
<script lang="ts" setup>
    import { ref } from 'vue';

    // 变量名必须和模板中 ref 属性的值一致
    let myDiv = ref();

    function showDiv() {
        console.log(myDiv.value);  // 此时 myDiv.value 才有值
    }
</script>
```

**不同组件中同名 ref 不会互相干扰：** 每个组件都有自己独立的 setup 函数域。

### 8.4.2 组件标签上使用 ref（获取子组件实例）

```vue
<!-- 父组件 App.vue -->
<template>
    <div>
        <Person ref="personInstance"></Person>
        <button @click="showPersonInstance">输出 Person 组件实例</button>
    </div>
</template>
<script lang="ts" setup>
    import { ref } from 'vue';
    import Person from './components/Person.vue';

    // personInstance 代表 Person 组件实例（Proxy 代理对象）
    let personInstance = ref();

    function showPersonInstance() {
        console.log(personInstance.value);
        console.log('Person 组件实例上的数据', personInstance.value.a);
    }
</script>
```

### 8.4.3 defineExpose（子组件暴露数据给父组件）

子组件使用 `defineExpose` 对外暴露数据或函数，父组件通过子组件实例对象访问：

**子组件：**

```vue
<script lang="ts" setup>
    import { ref } from 'vue';

    let a = ref(100);

    function greet() {
        console.log('Hello from child!');
    }

    // 暴露给父组件（可以暴露数据，也可以暴露函数）
    defineExpose({ a, greet });
</script>
```

**父组件：**

```vue
<script lang="ts" setup>
    import { ref } from 'vue';

    let personInstance = ref();

    function showData() {
        console.log('Person 组件实例上的数据：', personInstance.value.a);
        personInstance.value.greet();  // 调用子组件暴露的方法
    }
</script>
```

**总结：** 通过 `ref` + `defineExpose`，可以完成**子组件向父组件传递数据**的效果。

## 8.5 $attrs（祖→孙透传）

`$attrs` 用于实现**当前组件**的**父组件**向**当前组件**的**子组件**传递数据（**祖→孙**，通过它也可以完成**孙→祖**）。

- `$attrs` 是一个对象，包含所有父组件传入的标签属性。
- `$attrs` 会自动排除 `defineProps` 中声明的属性。

**父组件：**

```vue
<template>
    <div class="parent">
        <h1>父组件</h1>
        <h2>a: {<!-- -->{ a }}</h2>
        <h2>b: {<!-- -->{ b }}</h2>
        <!-- v-bind="{x: 300, y: 400}" 将对象的所有属性绑定到 Son 标签上 -->
        <Son :a="a" :b="b" v-bind="{ x: 300, y: 400 }" :changeB="changeB"/>
    </div>
</template>
<script lang="ts" setup>
    import { ref } from 'vue';
    import Son from './Son.vue';

    let a = ref(100);
    let b = ref(200);

    function changeB(val: number) { b.value += val; }
</script>
```

**子组件（中间人，不被打扰）：**

```vue
<template>
    <div class="son">
        <h2>子组件</h2>
        <h2>a: {<!-- -->{ a }}</h2>
        <h2>$attrs 中的数据：{<!-- -->{ $attrs }}</h2>
        <!-- 将 $attrs 透传给孙组件 -->
        <Grandson v-bind="$attrs"/>
    </div>
</template>
<script lang="ts" setup>
    import Grandson from './Grandson.vue';
    // a 在这里通过 defineProps 消费掉了，所以 $attrs 中还剩 b、x、y
    defineProps(['a']);
</script>
```

**孙组件：**

```vue
<template>
    <div class="grandson">
        <h2>孙组件</h2>
        <h2>b: {<!-- -->{ b }}</h2>
        <h2>x: {<!-- -->{ x }}</h2>
        <h2>y: {<!-- -->{ y }}</h2>
        <!-- 通过调用 changeB 函数，完成 孙→祖 的数据传递 -->
        <button @click="changeB(10)">修改爷爷组件中的 b</button>
    </div>
</template>
<script lang="ts" setup>
    defineProps(['b', 'x', 'y', 'changeB']);
</script>
```

## 8.6 $refs 与 $parent

- `$refs`：内置的，可以直接用，可以获取被 `ref` 属性标识的所有 DOM 元素或组件实例（对象形式）。
- `$parent`：内置的，可以直接用，可以在当前组件中获取父组件实例对象。

**父组件：**

```vue
<template>
    <div class="parent">
        <h1>父组件</h1>
        <h4>账户余额：{<!-- -->{ balance }}</h4>
        <button @click="giveFirst(1000)">给老大 1000</button>
        <button @click="giveSecond(2000)">给老二 2000</button>
        <!-- $refs 是内置的，包含所有被 ref 属性标识的 DOM 元素或组件实例 -->
        <button @click="giveFirstAndSecond($refs, 3000)">分别给老大和老二 3000</button>
        <Son1 ref="s1"></Son1>
        <Son2 ref="s2"></Son2>
    </div>
</template>
<script lang="ts" setup>
    import { ref } from 'vue';
    import Son1 from './Son1.vue';
    import Son2 from './Son2.vue';

    let balance = ref(10000);
    let s1 = ref();
    let s2 = ref();

    function giveFirst(value: number) {
        balance.value -= value;
        s1.value.balance += value;  // 修改子组件暴露的数据
    }
    function giveSecond(value: number) {
        balance.value -= value;
        s2.value.balance += value;
    }
    function giveFirstAndSecond(refs: any, value: number) {
        balance.value -= value * 2;
        for (const key in refs) {
            refs[key].balance += value;  // 遍历所有子组件
        }
    }

    defineExpose({ balance });  // 对外暴露数据，供子组件通过 $parent 访问
</script>
```

**子组件：**

```vue
<template>
    <div class="son">
        <h2>子组件1</h2>
        <h4>账户余额：{<!-- -->{ balance }}</h4>
        <!-- $parent 是内置的，获取当前组件的父组件实例 -->
        <button @click="giveAll($parent)">把钱都给父亲</button>
    </div>
</template>
<script lang="ts" setup>
    import { ref } from 'vue';

    let balance = ref(2000);

    function giveAll(parent: any) {
        parent.balance += balance.value;  // 修改父组件暴露的数据
        balance.value = 0;
    }

    defineExpose({ balance });  // 对外暴露数据，父组件通过 $refs 访问
</script>
```

## 8.7 provide 与 inject（祖←→孙）

通过 `provide` 和 `inject` 可以完成**祖←→孙**数据传递，且**中间组件（子组件）不被打扰**（与 `$attrs` 的区别：`$attrs` 会打扰中间人）。

**祖组件（使用 provide 提供数据）：**

```vue
<template>
    <div class="parent">
        <h1>父组件</h1>
        <h4>账户余额：{<!-- -->{ balance }}元</h4>
        <h4>房子：面积{<!-- -->{ house.area }}平方米，位置：{<!-- -->{ house.location }}</h4>
        <Son/>
    </div>
</template>
<script lang="ts" setup>
    import { provide, reactive, ref } from 'vue';
    import Son from './Son.vue';

    let balance = ref(10000);
    let house = reactive({ area: 100, location: '北京海淀X小区' });

    function changeBalance(value: number) { balance.value -= value; }

    // 提供数据
    // 注意：千万不要 balance.value，.value 传递过去的数据就不是响应式的了
    provide('balance', balance);
    provide('house', house);
    provide('changeBalance', changeBalance);
</script>
```

**中间子组件（不被打扰）：**

```vue
<template>
    <div class="son">
        <h2>子组件（中间人，不被打扰）</h2>
        <Grandson/>
    </div>
</template>
<script lang="ts" setup>
    import Grandson from './Grandson.vue';
    // 子组件不需要任何 props 或 $attrs，不被打扰
</script>
```

**孙组件（使用 inject 注入数据）：**

```vue
<template>
    <div class="grandson">
        <h3>孙组件</h3>
        <h4>爷爷的钱我继承了：{<!-- -->{ balance }}元</h4>
        <h4>爷爷的房子我继承了：面积{<!-- -->{ house.area }}平方米，位置：{<!-- -->{ house.location }}</h4>
        <button @click="changeBalance(100)">我要花爷爷 100 元</button>
    </div>
</template>
<script lang="ts" setup>
    import { inject } from 'vue';

    // 注入数据（第二个参数是默认值，用于解决 TS 类型推断问题）
    let balance = inject('balance', ref(0));
    let house = inject('house', { area: 0, location: '' });
    let changeBalance = inject('changeBalance', (x: number) => x);
</script>
```

## 8.9 插槽（Slots）

当父组件需要向子组件**动态注入自定义内容或结构**（如 HTML 片段、组件等）时，使用插槽（`<slot>`）。

**常见场景：** 子组件作为**布局容器**（如弹窗、卡片），父组件控制内部内容。

### 8.9.1 默认插槽

**子组件：**

```vue
<template>
    <div class="category">
        <h2>{<!-- -->{ title }}</h2>
        <!-- 默认插槽：父组件传入的内容会放到这里 -->
        <slot></slot>
    </div>
</template>
<script lang="ts" setup>
    defineProps(['title']);
</script>
<style scoped>
    .category {
        background-color: skyblue;
        border-radius: 10px;
        box-shadow: 0 0 10px;
        padding: 10px;
        width: 300px;
        height: 400px;
    }
    h2 { background-color: orange; text-align: center; }
</style>
```

**父组件：**

```vue
<template>
    <div class="parent">
        <h1>父组件</h1>
        <div class="content">
            <Category title="今日新闻热点">
                <!-- 这里的内容将会放到插槽上 -->
                <ul>
                    <li v-for="news in newsList" :key="news.id">{<!-- -->{ news.content }}</li>
                </ul>
            </Category>
            <Category title="今日美食推荐">
                <img :src="imgUrl" alt="美食图片加载失败">
            </Category>
            <Category title="今日影视推荐">
                <video :src="movieUrl" controls></video>
            </Category>
        </div>
    </div>
</template>
<script lang="ts" setup>
    import Category from './Category.vue';

    let newsList = [
        { id: 'news001', content: '澳总理发帖晒图：很荣幸参观中国长城' },
        { id: 'news002', content: '特朗普改口：无意炒掉美联储' },
        { id: 'news003', content: '巴西一11岁女孩拔牙时发现81颗牙齿' },
    ];
    let imgUrl = 'https://images.unsplash.com/photo-1568901346375-23c9450c58cd';
    let movieUrl = 'https://images-assets.nasa.gov/video/One%20Small%20Step/One%20Small%20Step~orig.mp4';
</script>
<style scoped>
    .content { display: flex; justify-content: space-evenly; }
    img, video { width: 100%; }
</style>
```

### 8.9.2 具名插槽

具名插槽：有名字的插槽，可以指定某段内容插入到某个指定的位置，更灵活。

**子组件：**

```vue
<template>
    <div class="category">
        <h2>{<!-- -->{ title }}</h2>
        <!-- 具名插槽：通过 name 属性命名 -->
        <slot name="s"></slot>
    </div>
</template>
<script lang="ts" setup>
    defineProps(['title']);
</script>
```

**父组件（三种语法）：**

```vue
<template>
    <div>
        <!-- 第一种：在组件标签上直接用 v-slot（不够灵活，整个标签内容都放到该插槽） -->
        <Category title="今日新闻热点" v-slot:s>
            <ul><li v-for="news in newsList" :key="news.id">{<!-- -->{ news.content }}</li></ul>
        </Category>

        <!-- 第二种：使用 template + v-slot（推荐，灵活指定内容） -->
        <Category title="今日美食推荐">
            <template v-slot:s>
                <img :src="imgUrl" alt="美食图片">
            </template>
        </Category>

        <!-- 第三种：使用 template + # 简写 -->
        <Category title="今日影视推荐">
            <template #s>
                <video :src="movieUrl" controls></video>
            </template>
        </Category>
    </div>
</template>
```

### 8.9.3 作用域插槽

**使用场景：** 数据在子组件当中，父组件在生成结构时需要使用子组件中的数据。

**子组件（通过插槽向父组件传递数据）：**

```vue
<template>
    <div class="news">
        <h2>今日热点新闻</h2>
        <!-- 作用域插槽：给父组件开了一个窗口，让父组件可以访问我的数据 -->
        <slot :newsList="newsList" x="100" y="200" name="s"></slot>
    </div>
</template>
<script lang="ts" setup>
    import { reactive } from 'vue';

    let newsList = reactive([
        { id: 'news001', content: '澳总理发帖晒图：很荣幸参观中国长城' },
        { id: 'news002', content: '特朗普改口：无意炒掉美联储' },
        { id: 'news003', content: '巴西一11岁女孩拔牙时发现81颗牙齿' },
    ]);
</script>
```

**父组件（接收子组件数据，自由决定渲染结构）：**

```vue
<template>
    <div>
        <!-- params 是子组件传递过来的对象，包含 newsList, x, y 等数据，名字不固定 -->
        <News>
            <template v-slot:s="params">
                <ul>
                    <li v-for="news in params.newsList" :key="news.id">{<!-- -->{ news.content }}</li>
                </ul>
            </template>
        </News>
        <News>
            <template v-slot:s="aaa">
                <ol>
                    <li v-for="news in aaa.newsList" :key="news.id">{<!-- -->{ news.content }}</li>
                </ol>
            </template>
        </News>
        <News>
            <template v-slot:s="params">
                <h5 v-for="news in params.newsList" :key="news.id">{<!-- -->{ news.content }}</h5>
            </template>
        </News>
    </div>
</template>
<script lang="ts" setup>
    import News from './News.vue';
</script>
```
