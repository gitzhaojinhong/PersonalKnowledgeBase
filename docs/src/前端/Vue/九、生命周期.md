

## 9.1 生命周期的理解

Vue3 的生命周期：指的是组件从**出生到死亡**的整个过程。

组件的生命周期通常包括四大阶段：

1. **创建**
2. **挂载**
3. **更新**
4. **销毁**

**为什么要学习生命周期？**

在一个组件的生命周期过程中，不同的时刻会去调用不同的函数（钩子函数）。如果我们需要在某个时间点上执行一段代码，可以把这段代码写到对应的钩子函数上，到了那个时间点就会被自动调用。

## 9.2 Vue2 的生命周期（对比参考）

4 个阶段，8 个钩子：

| 阶段 | 钩子函数 | 执行次数 |
| --- | --- | --- |
| 创建 | beforeCreate、created | 一次 |
| 挂载 | beforeMount、mounted | 一次 |
| 更新 | beforeUpdate、updated | 更新一次执行一次 |
| 销毁 | beforeDestroy、destroyed | 一次（组件销毁才触发） |

## 9.3 Vue3 的生命周期

4 个阶段，7 个钩子（以下讲解的是部分钩子，路由等还有其他钩子函数）：

| 阶段 | 钩子函数 | 执行次数 |
| --- | --- | --- |
| 创建 | 直接写到 setup 中的代码 | 一次 |
| 挂载 | onBeforeMount、onMounted | 一次 |
| 更新 | onBeforeUpdate、onUpdated | 更新一次执行一次 |
| 卸载 | onBeforeUnmount、onUnmounted | 一次（组件卸载才触发） |

**常用钩子（三个）：**

- `onMounted`：挂载完毕
- `onUpdated`：更新完毕
- `onBeforeUnmount`：卸载前

**完整示例：**

```vue
<template>
    <h3>计数器：{{ count }}</h3>
    <button @click="increment">计数器加 1</button>
</template>
<script lang="ts" setup>
    import {
        onBeforeMount, onMounted,
        onBeforeUpdate, onUpdated,
        onBeforeUnmount, onUnmounted,
        ref
    } from 'vue';

    let count = ref(0);
    function increment() { count.value++; }

    // 创建阶段（直接写在 setup 中的代码）
    console.log('创建');

    // 挂载前
    onBeforeMount(() => { console.log('挂载前'); });
    // 挂载后
    onMounted(() => { console.log('挂载后'); });
    // 更新前
    onBeforeUpdate(() => { console.log('更新前'); });
    // 更新后
    onUpdated(() => { console.log('更新后'); });
    // 卸载前
    onBeforeUnmount(() => { console.log('卸载前'); });
    // 卸载后
    onUnmounted(() => { console.log('卸载后'); });
</script>
```

**调试技巧：** 在需要停下来的位置编写 `debugger;`，可以在组件生命周期执行过程中设置断点。
