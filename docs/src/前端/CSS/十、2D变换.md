## 10.1 transform 概述

transform 属性用于对元素进行 2D 或 3D 变换。

```css
transform: transform-function(value);
```

## 10.2 2D位移

```css
/* 语法 */
transform: translate(x, y);

/* 示例 */
transform: translate(100px, 200px);   /* 向右移动 100px，向下移动 200px */
transform: translateX(100px);          /* 只向右移动 */
transform: translateY(200px);          /* 只向下移动 */
transform: translate(-50%, -50%);     /* 相对于自身尺寸 */
```

**特点：** 不会影响其他元素的位置。

## 10.3 2D缩放

```css
/* 语法 */
transform: scale(x, y);

/* 示例 */
transform: scale(2, 2);       /* 放大 2 倍 */
transform: scale(0.5, 0.5);    /* 缩小一半 */
transform: scaleX(2);         /* 只缩放宽度 */
transform: scaleY(2);         /* 只缩放高度 */
transform: scale(2);          /* 宽高都放大 2 倍 */
```

**注意：** scale 值可以是负数，表示翻转。

## 10.4 2D旋转

```css
/* 语法 */
transform: rotate(angle);

/* 示例 */
transform: rotate(45deg);     /* 顺时针旋转 45 度 */
transform: rotate(-45deg);    /* 逆时针旋转 45 度 */
```

**注意：** 正值顺时针，负值逆时针。

## 10.5 2D倾斜

```css
/* 语法 */
transform: skew(x, y);

/* 示例 */
transform: skew(30deg, 0deg);         /* 水平倾斜 30 度 */
transform: skewX(30deg);               /* 只水平倾斜 */
transform: skewY(30deg);               /* 只垂直倾斜 */
```

## 10.6 2D变换原点

```css
/* 语法 */
transform-origin: x y;

/* 示例 */
transform-origin: center center;      /* 默认，以中心为原点 */
transform-origin: left top;           /* 以左上角为原点 */
transform-origin: 0 0;                /* 同上 */
transform-origin: 50px 50px;          /* 以 (50px, 50px) 为原点 */
```

## 10.7 2D复合写法

```css
transform: translate(100px, 100px) rotate(45deg) scale(2);
```

**注意：** 变换的顺序会影响最终效果，建议将位移放在最前面。
