## 2.1 基本选择器

### 2.1.1 通配选择器

选中所有标签。

```css
* {
    margin: 0;
    padding: 0;
}
```

**应用场景：** 常用于清除默认样式（外边距、内边距）。

### 2.1.2 元素选择器

选中所有指定的元素（标签）。

```css
h2 {
    color: red;
}
```

### 2.1.3 类选择器

选中所有带有指定 class 属性的标签。

```css
.peiqi {
    color: blue;
}
```

```html
<h2 class="peiqi">我是一个h2标签</h2>
```

**特点：** class 属性的值可以重复，可以为一个标签设置多个 class 属性值。

### 2.1.4 ID选择器

选中带有指定 id 属性的标签。

```css
#peiqi {
    color: green;
}
```

```html
<h2 id="peiqi">我是一个h2标签</h2>
```

**特点：** id 属性值必须唯一，不能重复。

### 2.1.5 基本选择器总结

| 选择器 | 权重 | 说明 |
|--------|------|------|
| 通配选择器 | 0 | 选中所有标签 |
| 元素选择器 | 1 | 选中指定元素 |
| 类选择器 | 10 | 选中带有指定 class 的元素 |
| ID选择器 | 100 | 选中带有指定 id 的元素 |

**优先级：** ID选择器 > 类选择器 > 元素选择器 > 通配选择器

## 2.2 复合选择器

### 2.2.1 交集选择器

同时满足多个条件的元素。

```css
p.peqi {
    color: red;
}
```

**含义：** 必须是 `<p>` 标签，同时 class 属性值为 peqi。

### 2.2.2 并集选择器

同时选择多个选择器对应的元素。

```css
h2, .peiqi, #sixiang {
    color: red;
}
```

## 2.3 HTML元素间的关系

- **父元素：** 直接包裹当前元素的元素
- **子元素：** 被父元素直接包裹的元素
- **祖先元素：** 直接或间接包裹当前元素的元素（父元素是子元素的祖先）
- **后代元素：** 被祖先元素直接或间接包裹的元素（子元素是祖先元素的后代）
- **兄弟元素：** 同属于同一个父元素的元素

**关系：**
- 父元素 — 子元素（直接关系）
- 祖先元素 — 后代元素（间接关系）
- 兄弟元素（同级关系）

## 2.4 后代选择器

选中指定元素的所有后代元素。

```css
/* 选择 div 元素的所有后代 span 元素 */
div span {
    color: red;
}
```

```html
<div>
    <span>我是div的儿子span</span>
    <p>
        <span>我是div的孙子span</span>
    </p>
</div>
```

**示例说明：** 无论是直接子元素还是间接子元素，都会被选中。

## 2.5 子代选择器

只选中指定元素的直接子元素（儿子）。

```css
/* 选择 div 元素的直接子代 span 元素 */
div > span {
    color: red;
}
```

```html
<div>
    <span>我是div的儿子span</span>
    <p>
        <span>我是div的孙子span</span>
    </p>
</div>
```

**示例说明：** 只有直接子元素（儿子）会被选中，间接子元素（孙子）不会被选中。

## 2.6 兄弟选择器

### 2.6.1 相邻兄弟选择器

选中指定元素后面**紧邻**的兄弟元素。

```css
/* 选择 h1 后面紧邻的 span 元素 */
h1 + span {
    color: red;
}
```

### 2.6.2 通用兄弟选择器

选中指定元素后面**所有**的兄弟元素。

```css
/* 选择 h1 后面所有的 span 元素 */
h1 ~ span {
    color: red;
}
```

## 2.7 属性选择器

| 选择器 | 说明 |
|--------|------|
| `[属性名]` | 选中具有指定属性名的元素 |
| `[属性名="属性值"]` | 选中具有指定属性名和属性值的元素 |
| `[属性名^="开头值"]` | 选中属性值以指定开头值开头的元素 |
| `[属性名$="结尾值"]` | 选中属性值以指定结尾值结尾的元素 |
| `[属性名*="包含值"]` | 选中属性值包含指定值的元素 |

**示例：**

```css
/* 选中所有带有 disabled 属性的 input 元素 */
input[disabled] {
    width: 100px;
}

/* 选中所有 type 属性值为 text 的 input 元素 */
input[type="text"] {
    width: 100px;
}

/* 选中所有 class 属性值以 icon- 开头的 div 元素 */
div[class^="icon-"] {
    color: red;
}
```

## 2.8 伪类选择器

### 2.8.1 伪类概述

伪类是元素的一种特殊状态，格式为 `选择器:伪类 {}`。

### 2.8.2 链接伪类

| 选择器 | 说明 |
|--------|------|
| `:link` | 选中具有 href 属性的 a 链接（未访问） |
| `:visited` | 选中具有 href 属性的 a 链接（已访问） |
| `:hover` | 选中鼠标悬停的元素 |
| `:active` | 选中按下鼠标不松开的元素 |

**注意：** 如果需要同时生效，按照 LVHA 的顺序书写（link → visited → hover → active）。

### 2.8.3 元素状态伪类

| 选择器 | 说明 |
|--------|------|
| `:focus` | 选中获取焦点的 input 表单元素 |
| `:checked` | 选中被勾选的 radio 或 checkbox |
| `:disabled` | 选中被禁用的表单元素 |
| `:enabled` | 选中可用的表单元素 |

### 2.8.4 结构伪类

| 选择器 | 说明 |
|--------|------|
| `:first-child` | 选中父元素的第一个子元素 |
| `:last-child` | 选中父元素的最后一个子元素 |
| `:nth-child(n)` | 选中父元素的第 n 个子元素 |
| `:first-of-type` | 选中父元素中每种类型（标签）的第一个 |
| `:last-of-type` | 选中父元素中每种类型（标签）的最后一个 |
| `:nth-of-type(n)` | 选中父元素中每种类型（标签）的第 n 个 |

**n 的取值：**
- 数字：选中第 n 个
- 关键字 even：选中偶数
- 关键字 odd：选中奇数
- 公式：2n、2n+1、3n 等

**注意：** `:nth-child(n)` 先看位置再看类型，`:nth-of-type(n)` 先看类型再看位置。

### 2.8.5 否定伪类

```css
/* 排除 .hello 类的 li 元素 */
li:not(.hello) {
    color: red;
}
```

## 2.9 伪元素选择器

伪元素用于在文档中插入假想的元素，格式为 `选择器::伪元素 {}`。

| 选择器 | 说明 |
|--------|------|
| `::first-letter` | 选中第一个文字 |
| `::first-line` | 选中第一行文字 |
| `::before` | 在元素开头插入内容 |
| `::after` | 在元素末尾插入内容 |

**注意：** before 和 after 必须设置 content 属性才能生效。

**示例：**

```css
div::before {
    content: "开头";
    color: green;
}
div::after {
    content: "结尾";
    color: blue;
}
```

## 2.10 选择器的优先级（权重）

当多个选择器作用于同一个元素时，根据优先级判断哪个样式生效。

### 2.10.1 权重计算公式

权重计算公式：`Σ(行内样式, ID选择器, 类选择器, 元素选择器)`

| 选择器 | 权重 |
|--------|------|
| 行内样式 | 1000 |
| ID选择器 | 100 |
| 类选择器 | 10 |
| 元素选择器 | 1 |

### 2.10.2 权重叠加

当多个选择器叠加时，权重会累加。

```css
/* 权重：10 */
.hello {
    color: red;
}

/* 权重：100 */
#peiqi {
    color: blue;
}

/* 权重：100 + 1 = 101 */
#peiqi span {
    color: green;
}

/* 权重：100 + 10 = 110 */
#peiqi .sixiang {
    color: pink;
}
```

### 2.10.3 优先级判断技巧

1. 先比较权重之和，权重大的优先级高
2. 权重相同时，后写的样式优先级高（就近原则）
3. 权重无论如何都不可能超过行内样式的权重
4. `!important` 权重最高，但不推荐使用
