## 13.1 动画简介

动画是比过渡更强大的效果，可以实现更复杂的动画。

**基本步骤：**
1. 定义动画（@keyframes）
2. 使用动画（animation 属性）

## 13.2 @keyframes 规则

```css
@keyframes 动画名称 {
    from {
        /* 初始状态 */
        transform: translateX(0);
    }
    to {
        /* 结束状态 */
        transform: translateX(100px);
    }
}
```

**或使用百分比：**

```css
@keyframes 动画名称 {
    0% {
        transform: translateX(0);
    }
    50% {
        transform: translateX(50px);
    }
    100% {
        transform: translateX(100px);
    }
}
```

## 13.3 animation 属性

```css
/* 语法 */
animation: name duration timing-function delay count direction fill-mode play-state;

/* 示例 */
animation: move 1s ease-in-out infinite alternate;
```

## 13.4 animation 子属性

### 13.4.1 animation-name

指定动画名称。

```css
animation-name: move;
```

### 13.4.2 animation-duration

指定动画时长。

```css
animation-duration: 1s;
```

### 13.4.3 animation-timing-function

指定动画时间曲线。

```css
animation-timing-function: ease;
animation-timing-function: steps(4);
```

### 13.4.4 animation-delay

指定动画延迟。

```css
animation-delay: 0.5s;
```

### 13.4.5 animation-iteration-count

指定动画播放次数。

```css
animation-iteration-count: 1;          /* 播放 1 次 */
animation-iteration-count: infinite;   /* 无限循环 */
```

### 13.4.6 animation-direction

指定动画播放方向。

```css
animation-direction: normal;           /* 正常播放 */
animation-direction: reverse;          /* 反向播放 */
animation-direction: alternate;        /* 往返播放 */
animation-direction: alternate-reverse; /* 反向往返 */
```

### 13.4.7 animation-fill-mode

指定动画播放前后的状态。

```css
animation-fill-mode: none;             /* 默认，回到初始状态 */
animation-fill-mode: forwards;         /* 停留在结束状态 */
animation-fill-mode: backwards;        /* 在延迟期间显示初始状态 */
animation-fill-mode: both;             /* 同时应用 forwards 和 backwards */
```

### 13.4.8 animation-play-state

指定动画播放状态。

```css
animation-play-state: running;         /* 播放 */
animation-play-state: paused;          /* 暂停 */
```

**应用：** 可以配合 hover 实现鼠标悬停播放动画。

## 13.5 动画的注意事项

1. 动画不需要触发条件，可以自动播放
2. 可以通过 animation-play-state 控制播放/暂停
3. 建议将 transform 放在 @keyframes 的开始和结束状态
