## 12.1 过渡属性

transition 属性用于在元素状态变化时添加过渡效果。

```css
/* 语法 */
transition: property duration timing-function delay;

/* 示例 */
transition: all 0.5s ease-in-out 0s;
```

## 12.2 过渡的子属性

### 12.2.1 transition-property

指定要过渡的属性。

```css
transition-property: all;           /* 所有属性 */
transition-property: width;         /* 只过渡宽度 */
transition-property: width, height;  /* 过渡宽度和高度 */
```

### 12.2.2 transition-duration

指定过渡的持续时间。

```css
transition-duration: 0.5s;           /* 0.5 秒 */
transition-duration: 500ms;          /* 500 毫秒 */
```

### 12.2.3 transition-timing-function

指定过渡的时间曲线。

```css
transition-timing-function: linear;      /* 匀速 */
transition-timing-function: ease;        /* 先慢后快再慢（默认） */
transition-timing-function: ease-in;     /* 慢速开始 */
transition-timing-function: ease-out;     /* 慢速结束 */
transition-timing-function: ease-in-out; /* 慢速开始和结束 */
transition-timing-function: step-start;  /* 立即跳到结束 */
transition-timing-function: step-end;    /* 保持开始，直到结束 */
transition-timing-function: steps(4);   /* 分 4 步过渡 */
```

### 12.2.4 transition-delay

指定过渡的延迟时间。

```css
transition-delay: 0s;       /* 无延迟 */
transition-delay: 1s;       /* 延迟 1 秒 */
```

## 12.3 过渡的复合写法

```css
/* 完整写法 */
transition: width 0.5s ease-in-out 0s, height 0.5s ease-in-out 0s;

/* 简写 */
transition: all 0.5s ease-in-out;

/* 多个过渡 */
transition: width 0.5s, height 0.5s, background-color 0.5s;
```

## 12.4 过渡的注意事项

1. 过渡必须由状态变化触发（如 hover）
2. 过渡时长不宜过长，一般 0.3s-0.5s
3. 不是所有属性都能过渡（一般是有中间值的属性）
