## 11.1 3D位移

```css
/* 语法 */
transform: translate3d(x, y, z);
transform: translateX(100px);
transform: translateY(100px);
transform: translateZ(100px);
```

## 11.2 3D缩放

```css
transform: scale3d(x, y, z);
transform: scaleX(2);
transform: scaleY(2);
transform: scaleZ(2);
```

## 11.3 3D旋转

```css
/* 语法 */
transform: rotate3d(x, y, z, angle);

/* 示例 */
transform: rotateX(45deg);    /* 绕 X 轴旋转 */
transform: rotateY(45deg);    /* 绕 Y 轴旋转 */
transform: rotateZ(45deg);    /* 绕 Z 轴旋转（与 2D 旋转相同） */

/* 绕指定轴旋转 */
transform: rotate3d(1, 0, 0, 45deg);   /* 绕 X 轴 */
transform: rotate3d(0, 1, 0, 45deg);   /* 绕 Y 轴 */
transform: rotate3d(1, 1, 0, 45deg);   /* 绕 (1,1,0) 向量旋转 */
```

## 11.4 透视

perspective 属性用于设置观察者与 3D 元素的距离。

```css
/* 在父元素上设置 */
perspective: 1000px;

/* 或在子元素上设置 */
transform: perspective(1000px) rotateX(45deg);
```

**注意：**
- perspective 值越小，3D 效果越明显
- perspective 一般设置在父元素上

## 11.5 3D呈现

transform-style 属性用于设置子元素是否保持 3D 位置。

```css
transform-style: flat;         /* 默认，平面呈现 */
transform-style: preserve-3d;  /* 保持 3D 位置 */
```

## 11.6 3D变换原点

```css
transform-origin: x y z;
transform-origin: center center center;
transform-origin: 0 0 0;
```
