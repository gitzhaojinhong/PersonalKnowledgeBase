## 2.1 什么是HTML

**全称**：HyperText Markup Language（超文本标记语言）

- **超文本**：比普通文本更丰富，可以包含图片、链接、音视频、表格等内容
- **标记**：通过特定的标签符号，将普通文本转换为具有结构和语义的超文本
- **语言**：每个标签的写法、读音、使用规则，共同构成了一套标记语言

HTML不是编程语言，而是一种标记语言——它描述内容的结构和含义，但不具备逻辑运算能力。

## 2.2 相关国际组织

### 2.2.1 IETF

**全称**：Internet Engineering Task Force（国际互联网工程任务组）

成立于1985年底，是互联网技术标准化领域最具权威的组织，负责互联网相关技术规范的研发和制定。绝大多数国际互联网技术标准均出自IETF。

官网：https://www.ietf.org

### 2.2.2 W3C

**全称**：World Wide Web Consortium（万维网联盟）

创建于1994年，是Web技术领域最具影响力的标准制定机构。共发布了200多项技术标准和实施指南，对互联网技术的发展和应用起到了基础性支撑作用。

官网：https://www.w3.org

### 2.2.3 WHATWG

**全称**：Web Hypertext Application Technology Working Group（网页超文本应用技术工作小组）

成立于2004年，以推动HTML5标准为目的，由Opera、Mozilla基金会、Apple等浏览器厂商组成。WHATWG维护的HTML Living Standard是目前HTML事实上的最新标准。

官网：https://whatwg.org/

## 2.3 HTML发展历史

HTML从1.0逐步演进，经历了多个版本迭代：
- HTML 1.0 ~ 3.2：早期版本，功能简单
- HTML 4.01（1999年）：引入了CSS支持和脚本能力，是Web标准化的里程碑
- XHTML：试图将HTML严格化为XML语法，但未能普及
- HTML5（2014年正式发布）：引入了大量新标签、新API、多媒体能力，是当前的标准

目前HTML的标准由WHATWG以"Living Standard"的形式持续维护更新，不再以版本号区分。
