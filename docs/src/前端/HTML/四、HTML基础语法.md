## 4.1 标签概述

标签（又称元素）是HTML的基本组成单位，用于定义内容的结构和含义。

标签分为两种：
- **双标签**（绝大多数）：`<标签名>内容</标签名>`，内容写在开始标签和结束标签之间
- **单标签**（自闭合）：`<标签名>`，不需要结束标签

```html
<!-- 双标签示例 -->
<p>这是一个段落</p>

<!-- 单标签示例 -->
<br>
<hr>
<img src="photo.jpg" alt="照片">
<input type="text">
```

标签名不区分大小写（`<P>` 和 `<p>` 效果相同），但规范上推荐全部使用小写。

## 4.2 标签之间的关系

- **并列关系**：标签处于同一层级，互为兄弟关系
- **嵌套关系**：一个标签包含在另一个标签内部，形成父子关系

使用Tab缩进来体现嵌套关系，使代码结构一目了然：

```html
<div>
    <h2>标题</h2>      <!-- h2和p是并列关系 -->
    <p>段落内容</p>     <!-- h2和p都是div的子元素 -->
</div>
<div>
    <p>另一个段落</p>   <!-- 这个div和上面的div是并列关系 -->
</div>
```

## 4.3 标签属性

### 4.3.1 基本语法

属性用于给标签提供附加信息，写在起始标签或单标签中，格式为 `属性名="属性值"`。

```html
<input type="password" name="pwd" maxlength="20">
<a href="https://www.example.com" target="_blank">链接文字</a>
```

一个标签可以同时设置多个属性，属性之间用空格分隔，书写顺序不影响功能。

### 4.3.2 布尔属性

有些属性没有属性值（或属性值与属性名相同），这种称为布尔属性。布尔属性只要出现在标签中就表示"启用"，不写就表示"关闭"。

```html
<input disabled>           <!-- 禁用输入框 -->
<input type="radio" checked>  <!-- 默认选中 -->
<video controls autoplay muted>  <!-- 显示控件、自动播放、静音 -->
```

也可以写成显式赋值形式（效果相同）：
```html
<input disabled="disabled">
<input type="radio" checked="checked">
```

### 4.3.3 属性使用规范

1. 不同的标签有不同的专属属性；也有一些**通用属性**（如id、class、style、title等）适用于所有标签
2. 属性名、属性值都是W3C规范定义好的，不能随意编造
3. 属性名和属性值推荐使用小写
4. 属性值用双引号包裹（单引号也可以，甚至省略引号在多数情况下也行，但推荐使用双引号以避免歧义）
5. 同一标签中不要出现同名属性，后面的值会覆盖前面的值

```html
<!-- 错误：两个type属性，后者覆盖前者 -->
<input type="text" type="password">
<!-- 实际效果是 password -->
```

## 4.4 HTML注释

注释是对代码的解释和说明，会被浏览器忽略，不会显示在页面上，但在源代码中可见。

```html
<!-- 这是一段注释，不会显示在页面上 -->
<p>这是可见的内容</p>

<!-- 
    多行注释
    第二行
    第三行
-->
```

注释的典型使用场景：
- 解释复杂逻辑或特殊处理的原因
- 临时注释掉一段代码用于调试
- 标记代码区块的功能（如 `<!-- 导航栏开始 -->`）

> 注意：HTML注释不能嵌套，以下写法是错误的：
```html
<!--
    外层注释
    <!-- 内层注释 → 这是错误的！ -->
-->
```
