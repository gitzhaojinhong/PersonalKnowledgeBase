## 12.2 生产环境选型建议

| 场景 | 推荐方案 |
|:---|:---|
| **简单计数（低并发 < 1000）** | `AtomicInteger` |
| **高并发计数（≥ 1000）** | `LongAdder` |
| **普通同步（单写多读）** | `synchronized` |
| **需要 tryLock / 超时** | `ReentrantLock` |
| **读多写少（< 10:1）** | `ReentrantReadWriteLock` / `StampedLock` |
| **线程安全 Map（无序）** | `ConcurrentHashMap` |
| **线程安全 Map（有序）** | `ConcurrentSkipListMap` |
| **线程安全 List（读多写少）** | `CopyOnWriteArrayList` |
| **线程安全 Set（读多写少）** | `CopyOnWriteArraySet` |
| **生产者-消费者** | `ArrayBlockingQueue` / `LinkedBlockingQueue` |
| **延时任务队列** | `DelayQueue` |
| **线程直接交接** | `SynchronousQueue` |
| **等待多线程完成** | `CountDownLatch` |
| **多阶段多线程同步** | `CyclicBarrier` / `Phaser` |
| **限流** | `Semaphore` |
| **定时任务调度** | `ScheduledThreadPoolExecutor` |
| **异步任务链式编排** | `CompletableFuture` |
| **线程隔离变量** | `ThreadLocal`（记得 remove） |
| **高并发随机数** | `ThreadLocalRandom` |
| **线程阻塞/唤醒底层** | `LockSupport` |
