## 四、分页查询

### 4.1 分页插件配置

**MyBatis-Plus 3.5.x 推荐使用新版分页插件配置**：

```java
import com.baomidou.mybatisplus.annotation.DbType;
import com.baomidou.mybatisplus.extension.plugins.MybatisPlusInterceptor;
import com.baomidou.mybatisplus.extension.plugins.inner.PaginationInnerInterceptor;
import org.springframework.context.annotation.Bean;
import org.springframework.context.annotation.Configuration;

@Configuration
public class MybatisPlusConfig {
    
    @Bean
    public MybatisPlusInterceptor mybatisPlusInterceptor() {
        MybatisPlusInterceptor interceptor = new MybatisPlusInterceptor();
        
        // 添加分页插件（必须指定数据库类型）
        PaginationInnerInterceptor paginationInterceptor = new PaginationInnerInterceptor(DbType.MYSQL);
        paginationInterceptor.setMaxLimit(1000L);  // 单页最大 1000 条
        paginationInterceptor.setOverflow(true);   // 页码溢出时回到最后一页
        
        interceptor.addInnerInterceptor(paginationInterceptor);
        
        return interceptor;
    }
}
```

> **重要**：**分页插件必须配置数据库类型**（`DbType.MYSQL`、`DbType.POSTGRE_SQL` 等），否则分页 SQL 无法正常生成。

### 4.2 基础分页用法

```java
// 1. 无条件分页（查询第 1 页，每页 10 条）
Page<User> page = new Page<>(1, 10);
Page<User> result = userService.page(page);

System.out.println("当前页：" + result.getCurrent());
System.out.println("每页大小：" + result.getSize());
System.out.println("总记录数：" + result.getTotal());
System.out.println("总页数：" + result.getPages());
System.out.println("数据列表：" + result.getRecords());

// 2. 条件分页
Page<User> page2 = new Page<>(2, 20);
LambdaQueryWrapper<User> wrapper = Wrappers.lambdaQuery()
    .gt(User::getAge, 18)
    .orderByDesc(User::getCreateTime);

Page<User> result2 = userService.page(page2, wrapper);

// 3. Mapper 层直接分页
Page<User> page3 = new Page<>(1, 10);
Page<User> result3 = userMapper.selectPage(page3, 
    Wrappers.<User>lambdaQuery().eq(User::getStatus, 1)
);
```

### 4.3 分页对象 Page 详解

```java
// Page 构造方法
Page<T> page = new Page<>();
Page<T> page = new Page<>(long current);           // 指定当前页
Page<T> page = new Page<>(long current, long size); // 指定当前页和每页大小
Page<T> page = new Page<>(current, size, boolean searchCount); // 是否查询总记录数
Page<T> page = new Page<>(current, size, long total); // 直接传入总记录数（不执行 COUNT 查询）
```

**Page 核心属性**：

| 属性 | 说明 |
|------|------|
| `current` | 当前页码（从 1 开始） |
| `size` | 每页显示条数 |
| `total` | 总记录数 |
| `pages` | 总页数 |
| `records` | 当前页数据列表 |
| `orders` | 排序信息 |
| `optimizeCountSql` | 是否优化 COUNT SQL（默认 true） |
| `searchCount` | 是否进行 COUNT 查询（默认 true） |

### 4.4 自定义分页 SQL

当需要多表关联分页时，可在 Mapper 中自定义 SQL。

#### 4.4.1 定义 DTO

```java
@Data
public class UserOrderDTO {
    private Long userId;
    private String username;
    private Integer orderCount;
    private BigDecimal totalAmount;
}
```

#### 4.4.2 Mapper 接口

```java
public interface UserMapper extends BaseMapper<User> {
    
    // 自定义分页查询：查询用户及其订单统计
    Page<UserOrderDTO> selectUserOrderPage(
        Page<UserOrderDTO> page,
        @Param("minAge") Integer minAge
    );
}
```

#### 4.4.3 XML 映射

```xml
<select id="selectUserOrderPage" resultType="com.example.dto.UserOrderDTO">
    SELECT 
        u.id AS userId,
        u.username,
        COUNT(o.id) AS orderCount,
        COALESCE(SUM(o.amount), 0) AS totalAmount
    FROM t_user u
    LEFT JOIN t_order o ON u.id = o.user_id
    WHERE u.deleted = 0
      AND u.age >= #{minAge}
    GROUP BY u.id, u.username
</select>
```

#### 4.4.4 Service 调用

```java
Page<UserOrderDTO> page = new Page<>(1, 10);
Page<UserOrderDTO> result = userMapper.selectUserOrderPage(page, 18);
```

> **关键点**：自定义分页方法的第一个参数必须是 `Page` 对象，MyBatis-Plus 会自动拦截并注入分页逻辑。

### 4.5 分页最佳实践

```java
/**
 * 通用分页查询封装
 */
public <T> PageResult<T> queryPage(PageParam param, Wrapper<T> wrapper, 
                                   Function<Page<T>, Page<T>> queryFunction) {
    // 参数校验
    long current = Math.max(1, param.getPageNum());
    long size = Math.min(param.getPageSize(), 100);  // 限制最大页大小
    
    Page<T> page = new Page<>(current, size);
    Page<T> result = queryFunction.apply(page);
    
    return PageResult.of(result.getRecords(), result.getTotal(), 
                         result.getCurrent(), result.getPages());
}

// 使用示例
PageResult<User> result = queryPage(
    new PageParam(1, 20),
    Wrappers.<User>lambdaQuery().eq(User::getStatus, 1),
    p -> userMapper.selectPage(p, Wrappers.<User>lambdaQuery().eq(User::getStatus, 1))
);
```
