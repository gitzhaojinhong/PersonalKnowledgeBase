## 十、附录

### 10.1 完整配置参考

```yaml
mybatis-plus:
  # Mapper XML 位置
  mapper-locations: classpath*:/mapper/**/*.xml
  # 实体类包扫描
  type-aliases-package: com.example.entity
  # 枚举包扫描
  type-enums-package: com.example.enums
  # 全局配置
  global-config:
    db-config:
      id-type: auto              # 主键策略
      logic-delete-field: deleted
      logic-delete-value: 1
      logic-not-delete-value: 0
      table-prefix: t_           # 全局表前缀
      table-underline: true      # 表名使用下划线命名
      capital-mode: false        # 大写命名
      insert-strategy: not_null  # 全局插入策略
      update-strategy: not_null  # 全局更新策略
      select-strategy: not_null  # 全局 WHERE 条件策略
    banner: true                 # 控制台打印 banner
  # MyBatis 原生配置
  configuration:
    map-underscore-to-camel-case: true
    cache-enabled: false
    log-impl: org.apache.ibatis.logging.stdout.StdOutImpl
    default-enum-type-handler: org.apache.ibatis.type.EnumOrdinalTypeHandler
  # 扩展配置
  ext:
    # 自定义配置...
```

### 10.3 MybatisX 插件（IDE 辅助工具）

**MybatisX** 是 MyBatis-Plus 团队开发的 IntelliJ IDEA 插件，可大幅提升开发效率。

**核心功能**：
- **Mapper-XML 互跳**：点击左侧箭头快速在 Mapper 接口与 XML 之间切换
- **代码自动生成**：根据数据库表结构生成 Entity、Mapper、Service、Controller 代码
- **JPA 风格 SQL 提示**：在 Mapper 接口中写方法名自动生成 SQL（如 `findByUsernameAndAge`）
- **SQL 语法检查**：实时检测 XML 中的 SQL 语法错误

**安装方式**：

打开 IDEA → File → Settings → Plugins → Marketplace → 搜索 "MybatisX" → Install

> **注意**：MybatisX 是可选开发工具，不影响项目运行，仅提升开发体验。

### 10.4 常见问题排查

| 问题现象 | 可能原因 | 解决方案 |
|---------|---------|---------|
| `Invalid bound statement` | Mapper XML 未扫描到 | 检查 `mapper-locations` 配置 |
| 分页不生效 | 分页插件未配置 | 添加 `PaginationInnerInterceptor` |
| 逻辑删除查询不到 | 逻辑删除字段值不对 | 检查 `logic-delete-value` 配置 |
| 自动填充不生效 | `MetaObjectHandler` 未注册为 Bean | 添加 `@Component` 注解 |
| 枚举映射失败 | 未配置 `type-enums-package` | 在 yml 中添加枚举包扫描路径 |
| `tenant_id` 未附加 | 多租户插件未生效 | 检查插件配置顺序，分页插件应在多租户之后 |
