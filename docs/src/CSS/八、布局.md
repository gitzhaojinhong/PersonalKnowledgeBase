## 8.1 版心

版心是页面的主体内容区域，通常居中显示。

```css
.container {
    width: 1200px;
    margin: 0 auto;
}
```

## 8.2 常用布局名词

| 名词 | 说明 |
|------|------|
| 版心 | 页面的主体内容区域 |
| 头部（header） | 页面的顶部区域 |
| 导航（nav） | 导航区域 |
| 横幅（banner） | 页面的大图区域 |
| 内容区（content） | 页面的主要内容 |
| 侧边栏（aside） | 页面的侧边区域 |
| 底部（footer） | 页面的底部区域 |

## 8.3 重置默认样式

不同浏览器的默认样式可能不同，需要统一重置。

```css
* {
    margin: 0;
    padding: 0;
    box-sizing: border-box;
}

body {
    font-family: "Microsoft YaHei", sans-serif;
}

a {
    text-decoration: none;
    color: inherit;
}

ul, ol {
    list-style: none;
}

img {
    vertical-align: middle;
}
```
