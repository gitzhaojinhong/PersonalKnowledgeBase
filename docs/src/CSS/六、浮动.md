## 6.1 浮动简介

浮动的本意是实现文字环绕图片的效果。

```css
float: left;   /* 左浮动 */
float: right;  /* 右浮动 */
float: none;   /* 不浮动 */
```

**示例：**

```html
<img src="./demo.png" alt="">
<p>这里是文字内容...</p>
```

```css
img {
    float: left;
}
```

## 6.2 浮动特点

1. 浮动元素会脱离标准流（脱标）
2. 浮动元素可以向左或向右移动，直到遇到父元素边框或另一个浮动元素
3. 浮动元素不在占据原来的位置（脱离标准流）
4. 浮动元素后面的标准流元素会上移
5. 浮动元素具有行内块元素的特点（可以设置宽高）

## 6.3 浮动影响

### 6.3.1 对父元素的影响

父元素的高度会塌陷（变成 0），因为浮动元素不占据标准流位置。

### 6.3.2 对兄弟元素的影响

- 前面的浮动元素：正常显示
- 后面的浮动元素：可能会被前面的浮动元素覆盖
- 后面的标准流元素：会上移（被浮动元素挤开）

## 6.4 清除浮动

当父元素没有设置高度时，子元素浮动会导致父元素高度为 0，需要清除浮动。

### 6.4.1 额外标签法（不推荐）

在浮动元素末尾添加一个空标签。

```html
<div class="father">
    <div class="son1">son1</div>
    <div class="son2">son2</div>
    <div class="clear"></div>
</div>
```

```css
.clear {
    clear: both;
}
```

### 6.4.2 overflow 属性法

给父元素添加 overflow 属性。

```css
.father {
    overflow: hidden;
}
```

### 6.4.3 伪元素清除法（推荐）

使用伪元素替代额外标签。

```css
.clearfix::after {
    content: "";
    display: block;
    clear: both;
}

/* 兼容 IE6 */
.clearfix {
    *zoom: 1;
}
```

```html
<div class="father clearfix">
    <div class="son1">son1</div>
    <div class="son2">son2</div>
</div>
```

### 6.4.4 双伪元素清除法

```css
.clearfix::before,
.clearfix::after {
    content: "";
    display: table;
}
.clearfix::after {
    clear: both;
}
.clearfix {
    *zoom: 1;
}
```
