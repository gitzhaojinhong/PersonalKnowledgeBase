## 3.1 层叠性

当多个选择器作用于同一个元素时，如果作用的样式属性相同，则会发生样式冲突。

**处理机制：** 根据选择器的权重判断哪个样式生效，权重大的覆盖权重小的。

**示例：**

```css
h1 {
    color: red;   /* 被覆盖 */
    color: blue;  /* 生效 */
}
```

## 3.2 继承性

元素会自动继承父元素上设置的一些特定属性。

### 3.2.1 能继承的属性

- 文字相关属性（font-*、text-*、line-height、color 等）
- 与元素外观相关的属性（list-style 等）

### 3.2.2 继承的特点

- 继承的权重为 0，比所有选择器的权重都低
- 如果元素本身有样式，则不继承父元素的样式

### 3.2.3 继承失效的情况

```css
/* 强制继承 */
div {
    width: 100%;
}

/* div 的直接子元素 p 继承 div 的宽度 */
div > p {
    width: inherit;
}
```

## 3.3 优先级

### 3.3.1 优先级规则

| 规则 | 说明 |
|------|------|
| 先比较权重 | 权重大的优先级高 |
| 权重相同时 | 后写的样式优先级高 |

### 3.3.2 !important

`!important` 用于提升某个属性的权重到最高。

```css
h1 {
    color: red !important;
}
```

**注意：**
- `!important` 不能提升继承的权重
- `!important` 不能随意使用，慎重使用

### 3.3.3 优先级实战技巧

1. 判断选择器能否选中元素（选择器是否生效）
2. 计算权重（行内样式 > ID > 类 > 元素）
3. 权重相同时，后写的优先级高
