## 7.1 定位简介

定位用于精确控制元素的位置。

```css
position: static;       /* 静态定位（默认值） */
position: relative;     /* 相对定位 */
position: absolute;     /* 绝对定位 */
position: fixed;         /* 固定定位 */
position: sticky;        /* 粘性定位 */
```

定位后需要配合偏移量使用：

```css
/* 偏移量 */
top: 100px;     /* 距离顶部 */
bottom: 100px;  /* 距离底部 */
left: 100px;    /* 距离左侧 */
right: 100px;   /* 距离右侧 */
```

## 7.2 相对定位（relative）

相对于元素**自身**原来的位置进行定位。

```css
.box {
    position: relative;
    top: 100px;
    left: 100px;
}
```

**特点：**
- 不脱标，元素仍然占据原来的位置
- 定位后元素的层级会提升（覆盖其他元素）

**应用场景：** 元素位置微调。

## 7.3 绝对定位（absolute）

相对于**最近的已定位祖先元素**进行定位，如果没有已定位的祖先元素，则相对于 body 进行定位。

```css
.box {
    position: absolute;
    top: 100px;
    left: 100px;
}
```

**特点：**
- 脱标，不占据原来的位置
- 具有行内块元素的特点（可以设置宽高）

**应用场景：** 元素相对于某个祖先元素精确定位。

**子绝父相：** 子元素绝对定位，父元素相对定位（常用组合）。

## 7.4 固定定位（fixed）

相对于**浏览器窗口**进行定位。

```css
.box {
    position: fixed;
    top: 100px;
    right: 100px;
}
```

**特点：**
- 脱标，不占据原来的位置
- 固定在浏览器窗口的某个位置
- 具有行内块元素的特点

**应用场景：** 固定导航栏、固定侧边栏、回到顶部按钮等。

## 7.5 粘性定位（sticky）

元素在滚动时固定在某个位置。

```css
.box {
    position: sticky;
    top: 100px;
}
```

**特点：**
- 不脱标，元素仍然占据原来的位置
- 元素在滚动时表现为固定定位

**注意：** 需要指定 top、bottom、left、right 之一才能生效。

## 7.6 定位的层级

### 7.6.1 z-index 属性

设置元素的堆叠顺序，数值越大越靠上。

```css
.box1 {
    position: relative;
    z-index: 1;
}

.box2 {
    position: relative;
    z-index: 2;  /* 覆盖 box1 */
}
```

### 7.6.2 定位层级规则

1. z-index 值越大，层级越高
2. z-index 相同时，后写的元素层级越高
3. 定位元素的层级高于浮动元素和标准流元素

## 7.7 定位的特殊应用

### 7.7.1 绝对定位居中

```css
.box {
    position: absolute;
    left: 50%;
    top: 50%;
    transform: translate(-50%, -50%);
}
```

### 7.7.2 固定定位居中

```css
.box {
    position: fixed;
    left: 50%;
    top: 50%;
    transform: translate(-50%, -50%);
}
```
