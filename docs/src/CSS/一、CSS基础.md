## 1.1 CSS简介

CSS，全称 Cascading Style Sheets（层叠样式表），是一种标记语言，用于给 HTML 结构设置样式，例如文字大小、颜色、元素宽高等。

**核心思想：** HTML 搭建结构，CSS 添加样式，实现结构与样式的分离。

## 1.2 CSS的编写位置

### 1.1.1 行内样式

写在标签的 `style` 属性中，又称内联样式。

```html
<h1 style="color:red;font-size:60px;">欢迎学习CSS</h1>
```

**特点：**
- style 属性的值必须符合 CSS 语法规范，格式为 `名:值;`
- 只能控制当前标签的样式
- 不推荐大量使用，仅用于对当前元素添加简单样式

### 1.1.2 内部样式

写在 HTML 页面内部，将所有 CSS 代码单独放在 `<style>` 标签中。

```html
<head>
<style>
    h1 {
        color: red;
        font-size: 40px;
    }
</style>
</head>
```

**特点：**
- `<style>` 标签一般放在 `<head>` 标签中
- 样式可复用，代码结构清晰
- 缺点：多个 HTML 页面无法复用样式

### 1.1.3 外部样式

写在单独的 `.css` 文件中，随后在 HTML 文件中引入使用。

```css
/* style.css */
h1 {
    color: red;
    font-size: 40px;
}
```

```html
<head>
<link rel="stylesheet" href="./style.css">
</head>
```

**特点：**
- CSS 代码在单独的 `.css` 文件中
- HTML 文件中通过 `<link>` 标签引入
- 可实现多个 HTML 页面复用同一套样式
- 推荐使用

## 1.3 样式表的优先级

默认情况下，CSS 的优先级规则是：**就近原则**（谁离结构近，谁的优先级高）。

优先级从高到低依次是：

1. 行内样式（权重：1000）
2. ID 选择器（权重：100）
3. 类选择器（权重：10）
4. 元素选择器（权重：1）

**注意：**
- 优先级相同时，后面的样式会覆盖前面的样式（遵循"就近原则"）
- 可以使用 `!important` 强制提升优先级（不推荐滥用）

## 1.4 CSS语法规范

**格式：**

```css
选择器 {
    属性名: 属性值;
}
```

**示例：**

```css
h1 {
    color: red;
    font-size: 60px;
}
```

**规范：**
- 选择器与大括号之间保留一个空格
- 冒号后保留一个空格
- 每一行属性结束后保留分号

## 1.5 CSS代码风格

**风格一：紧凑风格（老式）**

```css
h1 { color: red; font-size: 40px; }
```

**风格二：展开风格（推荐）**

```css
h1 {
    color: red;
    font-size: 40px;
}
```
