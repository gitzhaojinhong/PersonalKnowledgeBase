## 14.1 基本用法

```css
column-count: 3;          /* 将内容分成 3 列 */
column-width: 200px;       /* 每列宽度 200px */
```

## 14.2 列间距

```css
column-gap: 20px;         /* 列间距 */
```

## 14.3 列边框

```css
column-rule: 1px solid red;  /* 列之间的分隔线 */
```

## 14.4 列高度

```css
column-fill: auto;           /* 默认，由内容决定 */
column-fill: balance;         /* 各列高度尽可能平衡 */
```

## 14.5 跨列

```css
column-span: all;            /* 跨所有列 */
column-span: 1;              /* 不跨列（默认） */
```

## 14.6 复合写法

```css
columns: 3 200px;            /* column-count column-width */
```
