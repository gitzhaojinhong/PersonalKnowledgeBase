## 16.1 媒体查询

### 16.1.1 媒体类型

| 类型 | 说明 |
|------|------|
| all | 所有设备 |
| screen | 屏幕设备 |
| print | 打印设备 |
| speech | 屏幕阅读器 |

### 16.1.2 媒体特性

```css
/* 语法 */
@media media-type and (media-feature) {
    /* CSS 规则 */
}

/* 示例 */
@media screen and (min-width: 768px) {
    body {
        background-color: red;
    }
}
```

**常用媒体特性：**
- width/height：视口宽度/高度
- min-width/min-height：最小宽度/高度
- max-width/max-height：最大宽度/高度
- orientation：portrait（竖屏）/landscape（横屏）

### 16.1.3 常用阈值

| 设备 | 宽度范围 | 常用阈值 |
|------|----------|----------|
| 手机 | < 768px | max-width: 767px |
| 平板 | 768px - 991px | min-width: 768px |
| 桌面 | 992px - 1199px | min-width: 992px |
| 大桌面 | >= 1200px | min-width: 1200px |

### 16.1.4 结合外部样式

```html
<!-- 方式一：直接在 HTML 中引入 -->
<link rel="stylesheet" href="./mobile.css" media="screen and (max-width: 767px)">
<link rel="stylesheet" href="./tablet.css" media="screen and (min-width: 768px) and (max-width: 991px)">
<link rel="stylesheet" href="./desktop.css" media="screen and (min-width: 992px)">

<!-- 方式二：在 CSS 中使用 @media -->
<style>
    /* 手机样式 */
    @media screen and (max-width: 767px) {
        /* ... */
    }

    /* 平板样式 */
    @media screen and (min-width: 768px) and (max-width: 991px) {
        /* ... */
    }

    /* 桌面样式 */
    @media screen and (min-width: 992px) {
        /* ... */
    }
</style>
```

## 16.2 响应式开发策略

### 16.2.1 移动优先

先编写移动端样式，然后逐步添加平板和桌面端的样式。

```css
/* 移动端样式（默认） */
body {
    background-color: red;
}

/* 平板及以上 */
@media screen and (min-width: 768px) {
    body {
        background-color: green;
    }
}

/* 桌面及以上 */
@media screen and (min-width: 992px) {
    body {
        background-color: blue;
    }
}
```

### 16.2.2 桌面优先

先编写桌面端样式，然后逐步添加平板和移动端的样式。

```css
/* 桌面端样式（默认） */
body {
    background-color: blue;
}

/* 平板及以下 */
@media screen and (max-width: 991px) {
    body {
        background-color: green;
    }
}

/* 移动端 */
@media screen and (max-width: 767px) {
    body {
        background-color: red;
    }
}
```
