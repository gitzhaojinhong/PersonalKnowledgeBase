## 七、过滤器与监听器

### 7.1 过滤器（Filter）

过滤器用于在请求到达Servlet之前或响应返回客户端之前进行处理。

#### 过滤器生命周期

```
初始化(init) → 过滤(doFilter) → 销毁(destroy)
```

#### 过滤器配置

```java
@WebFilter(
    filterName = "EncodingFilter",
    urlPatterns = "/*",                    // 过滤所有请求
    initParams = {
        @WebInitParam(name = "encoding", value = "UTF-8")
    }
)
public class EncodingFilter implements Filter {
    
    private String encoding;
    
    @Override
    public void init(FilterConfig config) throws ServletException {
        // 获取初始化参数
        encoding = config.getInitParameter("encoding");
        if (encoding == null) {
            encoding = "UTF-8";
        }
    }
    
    @Override
    public void doFilter(ServletRequest req, ServletResponse resp, FilterChain chain) 
            throws IOException, ServletException {
        
        HttpServletRequest request = (HttpServletRequest) req;
        HttpServletResponse response = (HttpServletResponse) resp;
        
        // 请求处理前
        request.setCharacterEncoding(encoding);
        response.setCharacterEncoding(encoding);
        
        // 放行到下一个过滤器或Servlet
        chain.doFilter(req, resp);
        
        // 响应返回后（可选处理）
    }
    
    @Override
    public void destroy() {
        // 资源释放
    }
}
```

#### 登录验证过滤器

```java
@WebFilter("/*")
public class LoginFilter implements Filter {
    
    // 白名单路径
    private static final String[] WHITE_LIST = {
        "/login", "/register", "/css/", "/js/", "/images/"
    };
    
    @Override
    public void doFilter(ServletRequest req, ServletResponse resp, FilterChain chain) 
            throws IOException, ServletException {
        
        HttpServletRequest request = (HttpServletRequest) req;
        HttpServletResponse response = (HttpServletResponse) resp;
        
        String path = request.getServletPath();
        
        // 检查是否在白名单
        for (String white : WHITE_LIST) {
            if (path.startsWith(white)) {
                chain.doFilter(req, resp);
                return;
            }
        }
        
        // 检查登录状态
        HttpSession session = request.getSession(false);
        if (session != null && session.getAttribute("user") != null) {
            chain.doFilter(req, resp);
        } else {
            response.sendRedirect(request.getContextPath() + "/login");
        }
    }
}
```

#### 过滤器链执行顺序

多个过滤器按配置顺序执行：

```
请求 → Filter1 → Filter2 → Servlet → Filter2 → Filter1 → 响应
```

**顺序规则：**
- 注解配置：按类名字母顺序
- web.xml配置：按声明顺序

### 7.2 监听器（Listener）

监听器用于监听Web应用中的事件，如应用启动/关闭、Session创建/销毁等。

#### 监听器类型

| 监听器接口 | 监听事件 | 用途 |
|------------|----------|------|
| `ServletContextListener` | 应用启动/关闭 | 初始化全局资源 |
| `ServletRequestListener` | 请求开始/结束 | 请求日志、性能监控 |
| `HttpSessionListener` | Session创建/销毁 | 在线人数统计 |
| `HttpSessionAttributeListener` | Session属性变化 | - |
| `ServletContextAttributeListener` | 上下文属性变化 | - |

#### 应用启动监听器

```java
@WebListener
public class AppStartupListener implements ServletContextListener {
    
    @Override
    public void contextInitialized(ServletContextEvent sce) {
        ServletContext context = sce.getServletContext();
        
        // 初始化数据库连接池
        DataSource dataSource = createDataSource();
        context.setAttribute("dataSource", dataSource);
        
        // 加载系统配置
        loadSystemConfig();
        
        System.out.println("应用启动完成");
    }
    
    @Override
    public void contextDestroyed(ServletContextEvent sce) {
        // 释放资源
        ServletContext context = sce.getServletContext();
        DataSource ds = (DataSource) context.getAttribute("dataSource");
        if (ds != null) {
            closeDataSource(ds);
        }
        
        System.out.println("应用关闭");
    }
}
```

#### Session监听器（在线人数统计）

```java
@WebListener
public class OnlineUserListener implements HttpSessionListener {
    
    private static final AtomicInteger onlineCount = new AtomicInteger(0);
    
    @Override
    public void sessionCreated(HttpSessionEvent se) {
        int count = onlineCount.incrementAndGet();
        se.getSession().getServletContext().setAttribute("onlineCount", count);
        System.out.println("Session创建，当前在线：" + count);
    }
    
    @Override
    public void sessionDestroyed(HttpSessionEvent se) {
        int count = onlineCount.decrementAndGet();
        se.getSession().getServletContext().setAttribute("onlineCount", count);
        System.out.println("Session销毁，当前在线：" + count);
    }
    
    // 获取在线人数
    public static int getOnlineCount() {
        return onlineCount.get();
    }
}
```
